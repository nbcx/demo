NB Framework

NB 是PHP语言开发的一套MVC框架，为WEB项目快速开发，高性能微服务开发提供简单好用的解决方案。

NB从设计之初，就为了兼容swoole和php-fpm两种方式，一套代码，可以根据业务需求在两种方式直接轻松切换。 我们提供了缓存，数据库，路由，数据验证等等所有开发中常用的功能，并且使其简单易用。

安装 Bootstrap

通过 Bower、Composer、Meteor 或者 npm 可以将 bootstrap 的 Sass 和 JavaScript 源码下载到本地。这些包管理工具所下载的 Bootstrap 文件中不包括文档，但是包含了完整的编译系统和说明文件。
